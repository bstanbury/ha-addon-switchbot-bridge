# SwitchBot Direct API Bridge — HA Add-on

Bypasses the SwitchBot Cloud HA integration's polling lag with direct API queries every 30 seconds. Provides real-time motion sensor states, lock/door status, and direct device commands.

## Installation
1. In HA: Settings → Add-ons → Add-on Store → ⋮ → Repositories
2. Add: `https://github.com/bstanbury/ha-addon-switchbot-bridge`
3. Install "SwitchBot Direct API Bridge"
4. Configure token and secret (from SwitchBot app → Developer Options)
5. Start
